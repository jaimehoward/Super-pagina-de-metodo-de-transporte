¿Que es el metodo de Costo Minimo?

El metodo de Costo Minimo es un algoritmo desarrollado con el objetivo de resolver problemas de transporte o distribucion.

¿Como se usa?

El metodo de Costo Minimo es muy sencilo de aplicar.

Se debe identificar el camino de menor costo de todas las columnas y filas y asignarle la mayor cantidad de producto disponible, una vez terminado este proceso, se debe repetir hasta cumplir con la demanda.
