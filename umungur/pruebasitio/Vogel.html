¿Que es el metodo vogel?

El metodo de vogel es un procedimiento euristico que tiene por objetivo minimizar costos de transporte, utiliza criiterios sencillos para la resolucion de problemas dificiles

¿Como se usa?

Para usar el metodo de vogel tendremos que determinar una penalizacion para cada fila y columna, esta penalizacion se crea a partir de la resta de los dos costos menores.

Se selecciona la fila o columna cuya penalizacion sea mas grande

Una vez elegida la fila o columna se aplica el metodo de menor costo
